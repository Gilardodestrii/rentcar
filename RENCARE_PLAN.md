# Rencar SaaS - Roadmap

## Stack
- Backend: Laravel 11
- Frontend: React + Inertia.js
- Styling: Tailwind CSS
- Database: MySQL
- Auth: Laravel Breeze

## Struktur Database (Draft)
- tenants (untuk SaaS multi-tenancy)
- users, roles, permissions
- cars (status, harga, specs, photos)
- drivers
- bookings (status, dates, total_price)
- expenses
- testimonials
- site_settings

## Langkah Awal
1. Setup project (Laravel + Breeze + Inertia)
2. Desain DB schema
3. Implementasi Landing Page (Public)
4. Implementasi Dashboard Admin
5. Subscription & Tenant logic
